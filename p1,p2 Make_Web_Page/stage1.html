CSS

CSS stands for Cascading Style Sheets,CSS describes how HTML elements are to be displayed on screen, paper, or in other media.CSS saves a lot of work. It can control the layout of multiple web pages all at once,External stylesheets are stored in CSS files.

Avoiding repetition

When programmers don't avoid repetition, they will often have to do the same thing over and over. This may mean copy and pasting certain STYLE attributes into many HTML elements re writing the same code (sometimes with minor differences) many times. This can lead to errors when the programmer decides to make a change to something. If they don't dilligently make that same change everywhere the repeated code appears, problems will arise.
